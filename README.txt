Mock USPTO Fusion Suite

How to run:
1. Unzip the folder.
2. Open index.html in a browser.
3. Best option: run a local static server.
   Example from the folder:
   python3 -m http.server 8000
   Then open http://localhost:8000

Pages included:
- index.html (Examination Console)
- analysis.html
- cases.html
- claim-chart.html
- office-action.html
- methodology.html
- about.html

Quick test:
- On the console page, click Load Sample Matter.
- Then go to Claim Chart and Office Action pages.
- The matter carries across pages through local browser storage.
